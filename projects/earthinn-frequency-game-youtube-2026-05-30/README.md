恐懼、憤怒、焦慮，每出現一次就是在消耗你自己心裡賬戶的餘額。
而現代人最大的問題就是，情緒賬戶長期處於透支狀態。
而認知行為學告訴我們了一個真相，事情本身並不會直接產生情緒，
真正產生情緒的是我們對事件的解讀。
同樣是會議被延後，有些人覺得打亂了節奏而焦慮，
有些人覺得多了一點準備的時間。
就像前面西博士說的，我們沒有辦法控制際遇，
但是我們可以掌控面對事情時的心態。
決定你能量透支的往往不是外界發生了什麼，
而是你怎麼去看待它。
聽到這裡可能有人會覺得，我懂了，
遇到不開心的事情就不要有負能量，不要有負面情緒是吧。
然後當他遇到了自己不高興的事情時，就拼命的壓抑，假裝沒事。
如果你也這麼做，這個道理你只認識了一半。
因為這裡還有一個多數人都會忽略的關鍵，
就是能量絕對不能夠被壓制，
而是要讓它流動，流動就代表著順暢與調節，
如果強行的壓抑它讓它停滯，反而會造成更嚴重的持續消耗。
我們每一種情緒都是身體自帶的信號，
如果你長期把這些信號硬壓下去，不去面對，
那這些負面的能量不會憑空的消失，
而是會深深的卡在你的身體裡造成更嚴重的持續消耗。
這個時候讓我們回過頭來看看西博士教給我們的方法，去攪動局面，
主動的開口聊天，表達感謝，保持好奇心。
表面上看，這些好像只是社交訊息，
但是往最底層去看，它根本就是在幫你做一件事。
把你從自我內耗封閉的狀態裡拉出來，
變成了一個開放的狀態，讓能量重新在你的身上流動起來。
你會發現，人與人之間的真誠的連接，
本質上就是一種美好的能量流動。
美國著名的精神科醫師大衛·霍金斯，
曾經就提出了一個非常有名的意識能量層級理論。
他發現，我們不同的情緒，就對應著不同的能量頻率。
如果從中間畫出一條線，上面是勇氣、寬恕、愛與慈悲等高能量狀態，
它是向外擴張的，
而下面則是憤怒、恐懼、羞愧等低能量狀態，它是向內收縮的。
仔細看這張表，上面的那些高能量的特質，
不就正好對應了西博士所說的，那跟支撐這艘帆船的核心價值嗎?
核心價值就是你自己內在的穩定系統，
當你清楚了自己想要成為什麼樣的人，
你就有了底氣，能夠穩穩的停留在那些高能量區域，
就不容易被外界的風浪拉進恐懼或者是內耗裡。
當能量收縮時，你就會進入防禦模式，
覺得做什麼都很卡，全世界都在跟你作對。
相反的，當能量擴張時流動起來了，你就會進入連結模式，
生活中的好機會自然就會發生，
這不是外在的世界變好了，而是你的狀態變了，
你跟世界互動的方式就會完全不同了。
所以如果你天天被這種負面的情緒帶著走，
這種長期的低能量就會轉變成所謂的「壞運氣」。
而當你由衷的去幫助別人，表達感謝你內在是打開的，
能量就會形成一種良性的循環回到你的身上，
這就是好運氣的真相。
也就是說，接住好運根本就不是什麼運氣技巧，
它純粹就是內在狀態的調整。
宇宙的本質是能量與振動，
你想要改變運氣最快的方法就是改變自身的頻率。
當你帶著滿格的電量去面對生活，
你就會發現問題並沒有縮小，而是你內在變強大了。
在這股看不見的能量交織中，
掌控好內在的狀態就是掌舵好船的方向。
無論航行中遇到什麼樣的風浪，
我們都能夠乘風破浪，抵達內心深處向往的那個目的地。
當然，航行久了，總會有能量見底，覺得疲憊的時候，
這再正常不過了，但也不要因為偶爾的低能量而感到焦慮，
給自己一點時間喘口氣，
因為我們每一個人都有自己重新調頻的超能力。
