# 🗼 Tokyo Trip Planner — AI 協作旅行規劃工具

> **目標：** 做一個互動式網頁，讓 Jones 和朋友兄弟一起規劃 2026 年 5 月初的東京旅行。

## 💡 核心概念

不是傳統的行程表 app，而是一個**卡片式靈感收集 + 日期軸拖放**的協作工具。

### 使用流程

1. **輸入關鍵字清單** — Jones 和朋友各自把想去的地方、想吃的東西、想做的事丟進來
2. **Agent 自動生成卡片** — Jarvis 在背景（晚上/heartbeat 空閒時）根據關鍵字：
   - 爬旅遊資訊、Google Maps 評分、營業時間
   - 產生推薦理由、附近景點、交通方式
   - 每個關鍵字做成一張「卡」，放在候備清單
3. **拖放排程** — 把候備卡拖到對應日期的時間軸上
4. **持續豐富** — Agent 定期自動補充更多靈感卡（類似 loot_filter.json 的遠端更新概念）

## 🔧 技術方向（待定）

| 面向 | 可能方案 |
|------|----------|
| 前端 | React / Astro + DnD library (react-beautiful-dnd / dnd-kit) |

## 📋 初步想法

- 靈感來源：Jones 用 loot_filter.json 遠端控制 bot 的經驗 — Agent 持續產生內容，人類做最終決策

- Jones 自己去過很多次，可以直接標記「推薦」和「跳過」
