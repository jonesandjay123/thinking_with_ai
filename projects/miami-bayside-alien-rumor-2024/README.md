# 2024 邁阿密 Bayside Marketplace「外星人事件」調查報告

Date: 2026-06-24  
Topic: 2024-01-01 美國佛州邁阿密 Bayside Marketplace 大量警車出動後，社群傳出「8-10 英尺高外星人 / shadow aliens」的都市傳說。

## 快速結論

我查了英文、中文、西文來源後，判斷如下：

1. **大量警車出現是真的，而且視覺上非常誇張。** 這不是 AI 假影片或純編造，當晚 Bayside Marketplace 一帶確實有大規模警力、道路封閉、人群逃跑與混亂畫面。
2. **可查證的官方與地方媒體版本很一致：大約 50 名青少年在商場製造騷亂、打架、施放煙火，有人把爆炸聲誤認為槍聲/active shooter，導致警方以高風險事件規格出動。**
3. **「外星人」說法目前沒有可靠證據。** 沒有近距離影片、沒有一致且可驗證的目擊者、沒有醫療/死亡/軍方/機場停擺紀錄，主流媒體、地方媒體、AFP、PolitiFact、Univision、Factchequeado 等來源都把它歸類為謠言/誤讀。
4. **最常被說成外星人的影像，大多是低畫質遠景裡的陰影/人影。** 警方說是「一個人走路的影子」；Forbes 引用更清楚的對照影片，認為更像 2-3 個人或警員一起走過警車與商場之間。
5. **為什麼後續消息少？** 因為在可查證層面，後續就是一般青少年騷亂與逮捕案件，不是未知生物事件。外星人敘事的生命週期主要在社群平台、迷因、陰謀論頻道與後續名人提及中延續。

我的總判斷：**這件事最值得注意的不是「外星人可能性很高」，而是「一個真實且高視覺衝擊的公共安全事件，如何在幾天內被低畫質影像、巨大警力、UAP 熱潮、社群平台演算法與不信任感改造成都市傳說」。**

## 事件時間線

### 2024-01-01 晚上：約 8:30 前後

NBC Miami 與 Local 10 報導，警方在晚上約 8:30 前接獲 Bayside Marketplace 有大群青少年造成騷亂的通報。警方說這群青少年打架、施放煙火、造成商家損害，部分人可能拿著棍棒。Local 10 報導指出，商家表示新年假期類似狀況並非第一次發生。

NBC Miami 的原始地方報導提到：

- 約 50 名青少年涉入。
- 青少年施放很大聲的煙火，造成商場民眾恐慌。
- 有居民把混亂誤認為槍聲並通報 active shooter。
- 有些警員一開始以高風險槍擊事件處理，因此重裝備接近人群。
- Biscayne Boulevard 2nd 到 4th Streets 一度封路。
- 4 名 14-16 歲青少年被捕，指控包含 burglary of unoccupied conveyance、grand theft、battery、resisting officer without violence 等。

### 2024-01-02 至 2024-01-05

社群上出現大量空拍/遠景影片：警車排滿街道、群眾奔跑、警燈密集。這些影片本身是真的或至少與當晚警力回應相符，但後續被附上不同敘事。

很快出現幾種說法：

- 「警方不是為小孩打架來的，是為 8-10 英尺高生物來的。」
- 「媒體沉默、警方掩蓋。」
- 「黑色軍用直升機、機場停擺、停電」等延伸說法。
- 「孩子們在商場打開 portal」之類更網路迷因化的版本。

### 2024-01-05

謠言在 X / Instagram / TikTok 上爆量，CBS Miami、NBC News、Forbes、NBC Miami、PolitiFact 等開始集中處理。Miami Police 對多家媒體回應：

- 沒有外星人、UFO 或 ET。
- 沒有機場關閉。
- 沒有停電。
- 沒有隱瞞公眾。
- 大量警力是因為青少年拒絕離開、需要 citywide officers 做 crowd control。

PolitiFact 引用 Miami Police Instagram 影片中 Rafael Horta 警官的說法：網路上說的 8-10 英尺外星人，其實是人走路加上影子；他說可以確認當時 Bayside Marketplace 沒有外星人。

### 2024-07

CBS Miami 又報導 Khloe Kardashian 在節目中提到這件事，讓「shadow aliens」都市傳說再度被娛樂新聞帶起。這不是新證據，而是謠言的二次流行。

## 為什麼警車多到像電影？

Jones 直覺上覺得「怎麼可能只是小孩放鞭炮」，這個懷疑很合理。單看畫面，警車數量真的不像一般商場糾紛。

但地方媒體提供了幾個讓警力規模變合理的因素：

1. **疑似 active shooter 通報。** 煙火爆炸聲被誤認為槍聲，有居民通報疑似槍手。美國警方對 active shooter 的初始反應會非常激烈，尤其在擁擠商場。
2. **人數不是幾個小孩，而是約 50 名青少年。** 如果多人打架、奔跑、拒絕離開、進入商店造成損害，警方可能升級成群眾控制事件。
3. **現場在 downtown / tourist area。** Bayside Marketplace 是觀光與商業區，警方需要快速封路、疏散、保護商家與人群。
4. **citywide response。** Univision / Factchequeado / AFP 都提到警方或警官解釋，現場警員控制不住，啟動 citywide 等級回應，因此全市可用警力都往現場集中。
5. **社群影片壓縮了時間與空間。** 空拍或高樓視角會把多街區警力集中成一個畫面，看起來像「所有警車都為一個點出動」。

所以「警車太多」本身不是外星人證據；它比較像是：**疑似槍擊 + 大型群眾事件 + downtown 商場 + 新年夜 + 警方過度準備/高規格回應** 的疊加。

## 為什麼網路會覺得可疑？

### 1. 影像不成比例

社群看到的是「幾十輛警車」，警方說的是「青少年煙火/打架」。人腦會覺得規模不匹配，於是尋找更大的原因。

### 2. 低畫質影像很容易誘發模式辨識

Forbes 的 Matt Novak 指出，有些影片被放大/加強後反而產生更多 digital artifacts；人腦會在像素化影像中找形狀。這對「高大灰影」敘事很重要。

### 3. 「大家都有手機，為何沒有近距離影片？」變成雙向論證

懷疑派會說：如果真有 8-10 英尺外星人，為什麼沒人拍近距離？  
陰謀論派會反過來說：正因為沒有影片，所以一定被刪除。

後者在邏輯上不可證偽，也很難進一步驗證。

### 4. UAP 氣氛已經在 2023 被點燃

2023 年美國國會 UAP 聽證會、David Grusch、墨西哥「非人類遺骸」事件等，使社群對外星議題高度敏感。2024 年第一週出現「大量警車 + 灰影」素材，很容易接上那個敘事。

### 5. 社群平台的藍勾/流量激勵

Forbes 指出，一些 X 藍勾帳號推動說法，但 X 的付費藍勾不等於可靠來源。高刺激說法更容易得到轉發與觀看。

## 各類來源怎麼說

### 英文地方媒體 / 主流媒體

- **NBC Miami / Local 10 / CBS Miami**：都把原始事件描述為青少年騷亂、打架、煙火、誤報槍擊、4 名青少年被捕。
- **NBC News / Telemundo**：把事件放在社群錯誤資訊脈絡下，引用 Miami Police 的「No aliens / LOL」說法。
- **Forbes**：兩篇文章分別指出外星人說法缺乏證據、低畫質影像更像人影/警員、放大影片會導致視覺誤讀。

### Fact-check

- **PolitiFact**：評為 Pants on Fire，說警方不是因外星生物而出動，而是回應約 50 名青少年造成的騷亂。
- **AFP Fact Check**：稱 Miami police refute alien invasion hoax，指出影片顯示的是警方回應青少年搶掠/煙火/騷亂。
- **Factchequeado / Univision / Chequeado**：西文查核一致認定，外星人說法無根據，警方已澄清，影像中的「外星人」是人影/陰影或被加工誤導。

### 中文來源

- **央視網 / 環球網轉述**：基本採取「美媒報導 + 警方否認」角度，提到 50 名青少年、煙火、打鬥、有人誤報槍擊、60 多輛警車、4 名青少年被捕。
- **大紀元 / 新唐人 / 看雜誌等**：偏「未解之謎」節目敘事，會強調疑點、黑色直升機、警方過多、目擊者稱不沉默等，但沒有提供比英文地方媒體更強的可驗證證據。

### 網路 / 都市傳說來源

- **Cryptid Wiki、Reddit、旅館/行銷部落格、YouTube 類節目**：多把它作為「Bayside Marketplace Aliens」都市傳說保存。這類來源有文化觀察價值，但證據力低。

## 目前可確認與不可確認

### 可確認

- 2024-01-01 晚上 Bayside Marketplace 確有大型警力回應。
- 現場確有混亂、人群奔跑、封路。
- 地方媒體和警方紀錄指向青少年騷亂、煙火、疑似槍擊誤報。
- 約 4 名青少年被捕。
- 「外星人」說法在 2024-01-05 左右爆紅，並被多家媒體與 fact-check 查核。
- Miami Police 多次否認有外星人、機場關閉、停電或隱瞞。

### 未找到可靠證據

- 沒有找到近距離、清晰、可驗證的外星生物影片。
- 沒有找到多名具名目擊者、可交叉比對細節的一致證詞。
- 沒有找到醫療、死亡、軍方、FAA、機場或電力系統異常紀錄支持外星人版本。
- 沒有找到後續正式調查把案件從青少年騷亂升級為未知生物/國安事件。

### 仍可追的資料

如果要更硬核，可以做 FOIA / public records request：

- Miami Police 2024-01-01 Bayside Marketplace dispatch logs / CAD logs
- 911 call audio
- Bodycam footage
- Arrest reports full text
- Incident report
- Road closure / citywide response logs

但目前公開媒體資料已足夠支持一個初步判斷：**外星人版本缺乏可驗證支撐。**

## 為什麼沒有更多後續消息？

我認為有三個原因：

1. **法律/警方層面，它是一般 juvenile riot / battery / theft / resisting case。** 青少年案件通常後續公開資訊有限，尤其涉及未成年人。
2. **媒體層面，外星人說法在 1 月 5 日左右被快速查核完成。** 對主流媒體來說，沒有新證據就沒有延續報導價值。
3. **社群層面，它變成都市傳說，不是新聞事件。** 後續在 Reddit、YouTube、TikTok、名人節目裡反覆出現，但通常是重新剪輯同一批影片，而不是新材料。

換句話說：**不是完全沒有後續，而是後續分裂成兩條線。官方/新聞線結束在「青少年騷亂 + 謠言澄清」；網路文化線則繼續把它當成未解之謎。**

## 我的判斷

我不會用「只是小孩放鞭炮」這句話輕描淡寫，因為它確實太小看現場規模。比較準確的描述應該是：

**這是一場在新年夜、觀光商場、約 50 名青少年、煙火爆炸聲、疑似槍擊通報、商家損害與群眾失控疊加下，引發 citywide police response 的公共安全事件。**

這樣就比較能解釋為什麼警車多到離譜。

但若問「目前是否有可靠證據支持外星人？」我的答案是：

**沒有。**

更像是低畫質影像 + 大量警車 + 人類模式辨識 + 社群不信任 + UAP 熱潮共同製造的現代都市傳說。
