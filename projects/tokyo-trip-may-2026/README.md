# Tokyo Trip May 2026

這個資料夾是 Jones、學偉、學灝與旅伴們 2026 年 5 月東京行的基地文件。

用途：
- 保存旅途中 Jarvis 產生的行程筆記、候選店家、截圖與參考資料
- 作為 LINE 群組分享圖片的公開 HTTPS 圖片來源
- 記錄哪些內容已經被放進 Trip Planner，哪些仍是討論素材

## Repo-as-media workflow

LINE 對「公開 HTTPS 圖片 URL」的支援比本機檔案附件穩定。未來如果 Jarvis 產生截圖、地圖、整理圖或其他圖片，可以：

1. 把圖片放到本資料夾的 `assets/` 下。
2. commit + push 到 GitHub。
3. 使用 GitHub raw URL 或 jsDelivr URL 發到 LINE。

範例：

```text
https://raw.githubusercontent.com/jonesandjay123/thinking_with_ai/main/projects/tokyo-trip-may-2026/assets/youtube-frames/doublexu-tonchin-ikebukuro-2025-frame.jpg
```
