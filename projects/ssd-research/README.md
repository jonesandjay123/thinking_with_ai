### 關鍵概念

MacBook Air M2 的 USB-C 孔支援 **Thunderbolt/USB4 (40Gbps)**，但在 USB 3.2 模式下：
- ✅ 支援 USB 3.2 Gen2 (10Gbps) → 實際 ~1000MB/s
- ❌ **不支援** USB 3.2 Gen2x2 (20Gbps) → 降速為 Gen2 (~1000MB/s)
- ✅ 支援 Thunderbolt 3/4 SSD → 但太貴，1080p 不需要

**結論：買 Gen2 的 SSD 就好。Gen2x2 的 SSD 在 Mac M2 上無法跑出 2000MB/s 的優勢，花更多錢卻得到相同速度。除非你未來打算換 Windows PC 或新 Mac，否則不值得。**
