# 🎮 Godot + AI 遊戲開發

> **目標：** 用 Godot（開源遊戲引擎）+ AI 輔助，零編程背景也能做出遊戲。

## 💡 靈感來源

- YouTube 上有人完全沒有編程背景，靠 Godot + AI 就完成了遊戲開發
- Godot 是開源免費的遊戲引擎，使用 GDScript（類 Python 語法）
- AI 可以幫忙寫遊戲邏輯、設計關卡、生成素材

## 🔍 研究方向

- [ ] 適合小型獨立遊戲的類型（2D platformer、puzzle、roguelike...）
- [ ] AI 生成遊戲美術素材的方案
- [ ] Godot + Vibe Coding 的可能性
