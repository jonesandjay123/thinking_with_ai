# Thinking with AI (與 AI 共同思考)

本 Repository 專門用來存放我與 AI (Deepmind Gemini) 共同探討各種議題的對話精華與重點總結。

## 🎯 目的 (Purpose)

透過將與 AI 的深度對話轉化為結構化的文檔，將稍縱即逝的靈感轉化為可執行的計畫與永久的知識資產。這不僅是紀錄，更是思考的延伸與行動的起點。

## 📂 結構 (Structure)

本 Repo 將根據每次討論的 **主題 (Topics)** 設立專屬的子資料夾，每個資料夾代表一個獨立的思考專案。
